# TextRPG Sparta

## Description

콘솔 환경에서 작동하는 간단한 RPG 게임입니다.

JSON 파일로 아이템 관리 및 캐릭터 정보를 저장하고, 싱글톤 SceneManager를 통해 씬 전환을 구조화했습니다.

## Technologies

C# (.Net 8.0)

Console Application

Newtonsoft.Json

## Main Features

상태보기: 캐릭터 능력치 및 장비 확인

인벤토리: 장비 장착 및 해제

상점: 아이템 구매 및 판매

던전: 도전 후 난이도 능력치 비례 보상

휴식: 체력 회복
